EcoSphere – Smart Sustainability Tracker
EcoSphere is a Streamlit-based web application that helps users track their daily lifestyle habits and measure their environmental impact. It provides insights, goals, and analytics to encourage a more sustainable way of living.
Features
Login & Signup
Track carbon, waste, energy
Dashboard with graphs
Set & track goals
Weather info
Eco score prediction
